# Lean SaaS Legal Checklist (Tailored to Soulvision LLC)

## 🧱 Business Structure

- [ ] Form LLC (NJ or NY)
- [ ] Operating Agreement (even single-member)
- [ ] Separate business bank account
- [ ] EIN from IRS
- [ ] Register domain under company (not personal)

## 📄 Required Website Documents

- [x] Terms of Service (Live in modal)
- [x] Privacy Policy (Live in modal)
- [x] Cookie Policy (Banner active)
- [x] AI Output Disclaimer section (In TemplatesView)
- [x] Copyright notice in footer
- [ ] DMCA policy page
- *Note: If serving CA users → comply with California Consumer Privacy Act*
- *Note: If serving EU users → comply with General Data Protection Regulation*

## 🔐 Data + Security

- [ ] HTTPS (SSL active)
- [ ] Encrypted passwords (bcrypt or Argon2)
- [ ] No storage of raw credit card data
- [ ] Stripe or compliant processor
- [ ] Database access restrictions
- [ ] Daily backups
- [ ] Rate limiting
- [ ] Admin logging enabled
- *Note: If handling payments → PCI standards from Payment Card Industry Security Standards Council*

## 🧠 AI-Specific Clauses

**Soulvision LLC MUST include:**

- [x] AI outputs may be inaccurate
- [x] Not professional advice
- [x] User responsible for implementation
- [x] You can monitor and remove abuse (in ToS)
- [x] No reverse engineering AI systems (in ToS)
- [x] No using outputs to train competing models (in ToS)

## 💰 Billing + Subscription

- [ ] Auto-renewal disclosure
- [x] Refund policy
- [ ] Cancellation method clearly stated
- [ ] Pricing page clarity
- [ ] Tax handling (sales tax if applicable)

## 🛡️ Risk Protection Clauses

**In ToS include:**

- [x] Limitation of liability (in ToS)
- [x] Indemnification (in ToS)
- [x] Arbitration clause (in ToS)
- [x] No class action (in ToS)
- [x] Governing law (NJ or NY) (in ToS)
- [x] Force majeure

## 🏷️ Intellectual Property

- [x] Copyright © Soulvision LLC 2026 (in Footer)
- [x] Define ownership of AI outputs (in ToS)
- [ ] Trademark filing (optional early, smart long term)
  - *File through United States Patent and Trademark Office*

# Legal Readiness Checklist Before Launch

### Stage 1 – MVP Private Beta (Safe to Launch)

- [ ] LLC formed
- [ ] Stripe integrated
- [x] ToS + Privacy Policy live
- [x] AI disclaimer visible
- [ ] HTTPS active

### Stage 2 – Public Launch

- [x] Arbitration clause included
- [x] Liability cap included
- [x] Refund policy clear
- [x] Cookie banner active
- [ ] Data deletion request process working
- [ ] Admin abuse monitoring

### Stage 3 – Scaling to Paying Users

- [ ] Cyber liability insurance
- [ ] Trademark filing
- [ ] Contractor IP agreements signed
- [ ] Incident response plan written
- [ ] Regular security audit

## 🧠 Strategic Advice

Launch lean but protected. Do not overpay for a lawyer before product-market fit.
**Do not launch without these 4 non-negotiables:**

1. Liability cap
2. AI disclaimer
3. Arbitration clause
4. LLC separation
